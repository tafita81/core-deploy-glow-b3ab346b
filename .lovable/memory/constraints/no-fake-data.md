---
name: No fake data
description: NEVER show simulated/fictitious data anywhere. Only real data from connected accounts. Show "Não conectado" when no data exists.
type: constraint
---
NUNCA inserir dados fictícios, simulados ou de exemplo em nenhum lugar do sistema.
Se não há conta conectada, mostrar "Não conectado" ou "R$ 0".
Se não há seguidores, mostrar 0 — nunca inventar números.
**Why:** O usuário exigiu explicitamente que tudo seja real. Dados falsos enganam e destroem confiança.
